# RepoAI

This repository contains a full-stack application with a Python backend and a TypeScript/Vite frontend.

- `backend/`: Python FastAPI backend and related agents
- `frontend/`: Vite + React frontend

## Setup

1. Install backend dependencies:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
2. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```

## Development

- Run backend from `backend/main.py`
- Run frontend with `npm run dev` in `frontend`
